# Welcome to PEKET

``peket``: **P**yCBC **E**xtension for **K**ilonova **E**mission **T**argeting.

This package serves as an add-on for PyCBC and NMMA/FIESTA. It is designed to perform coherent Gravitational Wave (GW) searches based on properties inferred from Kilonovae (KNe), and provides robust tools for generating and analyzing synthetic lightcurves.

Navigate through the sidebar to find installation instructions, detailed command-line tool references, and tutorials.
